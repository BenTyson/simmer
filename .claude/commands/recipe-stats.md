---
description: Show recipe database statistics
---

Query the database and display a statistics dashboard:

1. **Overview**
   - Total recipes
   - Total ingredients
   - Total instructions

2. **By Domain** (top 10)
   - Domain name
   - Recipe count
   - Percentage of total

3. **Scrape Queue Status**
   - Pending URLs
   - Completed URLs
   - Failed URLs

4. **Recent Activity**
   - Last 5 recipes added (name, domain, created_at)

Format as clean, readable sections.
